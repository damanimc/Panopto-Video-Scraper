# Panopto-Video-Scraper
Script to download all videos in folder from Panopto

## Installation

### 1. Get Tampermonkey
First, you need to install the Tampermonkey browser extension:

- **Chrome/Edge**: [Install from Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- **Firefox**: [Install from Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
- **Safari**: [Install from App Store](https://apps.apple.com/us/app/tampermonkey/id1482490089)
- **Opera**: [Install from Opera Add-ons](https://addons.opera.com/en/extensions/details/tampermonkey-beta/)

### 2. Install the Script
1. Once Tampermonkey is installed, click on the `panopto-scraper.user.js` file in this repository
2. Click the "Raw" button to view the raw script
3. Tampermonkey should automatically detect the userscript and prompt you to install it
4. Click "Install" to add the script to Tampermonkey

## Usage
1. Navigate to any Panopto folder with videos
2. The script will add a "Download All Videos" button to the page
3. Click the button to download videos from the folder
